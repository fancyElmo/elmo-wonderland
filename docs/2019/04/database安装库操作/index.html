MySQL安装

2019-04-05 入门知识 0 评论 字数统计:546字 阅读时长:2分

数据库安装

  • 在Linux终端输入如下指令 
    sudo apt-get install mysql-server
sudo apt-get install mysql-client
sudo apt-get install libmysqlclient-dev
  • 启动服务： sudo service myssql start
  • 查看进程中的MySQL服务: ps ajx|grep mysql
  • 停止服务： sudo service mysql stop
  • 重启服务： sudo service mysql restart
  • 查看数据库版本：select version()
  • 查看当前时间：select now()

常见参数

配置文件路径 /etc/mysql/mysql.conf.d/mysql.cnf

  • bind-address: 127.0.0.1-default
  • port: 3306-default
  • datadir(mysql存放路径): /var/lib/mysql

在终端操作数据库

  • 连接

mysql -u 用户名 -p

如果一切正确，就会连接到MySQL服务器，同时提示符变为 mysql>

  • 退出

quit 或者 exit

快捷键 crtl + d 也可以退出登录。注意，MySQL服务器仍在后台运行

  • 修改数据库密码

sudo mysqladmin -u 用户名 -p password

sudo service mysql restart

 

  • 忘记密码

1. 进入/etc/mysql/目录，并用root权限打开debian.cnf文件
cd /etc/mysql
sudo gedit debian.cnf

2. 进入mysql
mysql -u 用户名 -p
然后输入文件中的password值，即可进入mysql

3. 选择mysql数据库（用户名和密码均存储在此数据库的user表中）
 use mysql;

4. 显示user表中的列
show fields from user;
authentication_string这列就是密码

5. 修改密码
update mysql.user set authentication_string=password('123456') where user='root'

6. 退出
exit

7. 重启mysql
service mysql restart

 

  • 查看所有数据库

show databases;

注意 数据库操作语句结尾要写分号

  • 使用数据库

use 数据库名;

  • 创建数据库

creat database 数据库名 charset=utf8;

  • 删除数据库

drop database 数据库名;

卸载数据库

#sudo apt-get autoremove --purge mysql-server-5.7 
sudo apt-get remove mysql-server        
#sudo apt-get autoremove mysql-server  
sudo apt-get remove mysql-common
sudo rm -rf /etc/mysql/  /var/lib/mysql    #很重要
#清理残留数据
dpkg -l |grep ^rc|awk '{print $2}' |sudo xargs dpkg -P  
sudo apt autoremove
sudo apt autoreclean
