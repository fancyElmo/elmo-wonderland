安装MongoDB

2019-09-12 快照 0 评论 字数统计:426字 阅读时长:1分

1. 安装 MongoDB 社区版

Service Configuration 步骤中，使用默认的 Run service as Network Service user 即可。下方可以选择数据库和日志存储的地址，通用使用默认即可。

Install MongoDB Compass 步骤中，取消左下角勾选。MongoDB Compass 是Mongo自己的图形可视化工具，这里安装会默认到官网下载后安装，速度很慢。

等待安装完成…

配置环境变量 将 X:\..\MongoDB\Server\4.2\bin 添加到path内

2. 安装MongoDB Compass

3. 配置&启动服务

安装 Mongo 时已经配置过数据库和日志存储的地址，如果没有额外移动地址的需求，可以忽略配置这个步骤。

​ 在目标位置创建如下结构的文件夹及文件

-data
	-db
	-logs
		-mongodb.log

​ 编写配置文件

---X:\..\MongoDB\mongo.config---
# 数据库路径
dbpath=X:\..\data\db
# 日志输出文件路径
logpath=X:\..\data\logs\mongodb.log
# 错误日志采用追加模式
logappend=true
# 启用日志文件(默认启用)
journal=true
# 过滤无用日志信息(若需要调试请关闭)
quiet=true
# 端口号
port=27017

​ 配置本地 mongoDB 服务 ​ 管理员启动命令行 / 跳转至 ..\MongoDB\bin 目录 ​ 输入一下命令来根据上面的配置文件安装mongo服务

> mongod --config "X:\..\MongoDB\mongo.config" --install --serviceName "MongoDB"

​ 在 服务 中查看，出现 MongoDB 服务就证明成功了，通过命令行 net start MongoDB 就可以启动数据库了。一般设置了开机自启动后不再需要自己手动启动服务了。
