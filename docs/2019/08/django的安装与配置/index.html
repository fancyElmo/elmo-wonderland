Django的安装与配置

2019-08-02 入门知识 0 评论 字数统计:315字 阅读时长:1分

安装

  1. 安装Django包

    pip install django==2.1.4

  2. 检测是否安装以及查看Django版本

    python -m django --version

  3. 生成项目文件

     django-admin startproject 项目名
 # 命令执行后会在当前定位位置生成项目目录，且自动生成必要的文件
 ----django_site\

     |----db.sqlite3  # 内置数据库
     |----django_site\

     |    |----settings.py  # 配置文件
     |    |----urls.py  # 路由文件
     |    |----wsgi.py
     |----manage.py  # 入口函数
     |----templates\  # 模板
     |    |----login.html
     |    |----__init__.py
     |----user\  # 子项目
     |    |----admin.py  # 管理员模块
     |    |----apps.py
     |    |----migrations\  # 迁移
     |    |    |----__init__.py
     |    |----models.py  # 数据库模型
     |    |----tests.py  # 单元测试
     |    |----views.py  # 视图

  4. 生成django子项目

     python manage.py startapp 子项目名

  5. 启动服务

     python manage.py runserver

配置

  • settings.py 
      # 可访问地址 '*' -> 所有ip

  ALLOWED_HOSTS = []

  INSTALLED_APPS = [
      ...
  # 在此注册子应用

  'user',
  ]

  # 模板文件夹

  TEMPLATES = [
  {
      'BACKEND': 'django.template.backends.django.DjangoTemplates',
      # 根路由 模板文件夹名

      'DIRS': [os.path.join(BASE_DIR, 'templates')],
      ...
  }

  # 语言 汉语：zh-Hans

  LANGUAGE_CODE = 'zh-Hans'
  # 这里必须是 True，否则 LANGUAGE_CODE 会失效

  USE_I18N = True

  # 时区 国内时区:Asia/Shanghai

  TIME_ZONE = 'Asia/Shanghai'
  # 修改时区 True -> False

  USE_TZ = False
